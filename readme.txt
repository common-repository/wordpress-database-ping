"MySQL Server Has Gone Away" Plug-in Fix

== Description ==

Sometimes, under certain server configuration and plugins the wordpress website stop function corectlly and you keep getting a message that say "MySQL Server Has Gone Away".
This will fix this issue

== Installation ==

1. Upload `wordpress-database-ping` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin from Admin > Plugins menu.
3. Once activated you should check with Settings
